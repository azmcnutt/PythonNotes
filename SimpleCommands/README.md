## Update PIP

### Windows

  python -m pip install --upgrade pip

### Linux

  pip install --upgrade pip

## Create Virtual Environment

### Windows

  c:\Python35\python -m venv c:\path\to\myenv
  or
  python -m venv c:\path\to\myenv
  
### Linux

  python3 -m venv /path/to/new/virtual/environment
